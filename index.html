Versão Alternativa

O YouTube oferece diferentes qualidades de miniatura para seus vídeos.

Como Adicionar a Capa de um Vídeo do YouTube

  1. Obtenha o ID do vídeo do YouTube (ex: v6wWUlXdEDU da URL https://youtu.be/v6wWUlXdEDU)
  2. Use o formato: https://img.youtube.com/vi/VIDEO_ID/QUALIDADE.jpg
  3. Substitua QUALIDADE por:
    • maxresdefault.jpg - Maior resolução (1280x720)
    • hqdefault.jpg - Alta qualidade (480x360)
    • mqdefault.jpg - Qualidade média (320x180)
    • sddefault.jpg - Definição padrão (640x480)
  4. Coloque a imagem dentro de um link apontando para o vídeo no YouTube
  5. Adicione estilos CSS para melhorar a aparência e a interatividade
